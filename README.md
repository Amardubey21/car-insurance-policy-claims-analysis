Car Insurance Policy & Claims Analysis
Overview

This project analyzes car insurance policy sales and claim activity using simulated data. The goal is to understand premium revenue, claim costs, and portfolio risk using Python, SQL, and Power BI.

Tools Used

Python (Pandas, NumPy) – Data simulation

SQL (PostgreSQL) – Data analysis

Power BI – Dashboard visualization

Key Analysis

Total premium collected from policies sold in 2024

Monthly claim trends for 2025–2026

Claim cost vs premium by policy tenure

Portfolio loss ratio analysis

Policy distribution across tenures
